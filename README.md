# HealthOS — Curso INEMA

Curso aberto sobre o **HealthOS**: um coach de saúde pessoal num bot do Telegram,
aterrado nos seus dados (wearable, sangue, DNA, dieta), com Supabase + pgvector,
visão (Gemini), memória semântica e dashboard.

**3 trilhas · 12 módulos · 84 tópicos** — Fundamentos, Passo a passo e Como usar.
Página única self-contained (HTML + Tailwind via CDN), com camada de aprendizagem
(progresso, anotações, "minha jornada", temas), no padrão dark premium INEMA.CLUB.

- No ar: https://inematds.github.io/healthos/
- Portal: https://inema.club
- Blueprint de origem (código): https://github.com/earlyaidopters/health-os

> ⚠️ Conteúdo educacional. **Não é conselho médico.** Consulte profissionais de saúde.
